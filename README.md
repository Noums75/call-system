# call-system

Système d'appel simple pour FiveM basé sur ox_lib et bulletin.

## Dépendances

- [ox_lib](https://github.com/CommunityOx/ox_lib)
- [bulletin](https://github.com/Mobius1/bulletin)

## Utilisation

- `/appel` : ouvre le menu ox_lib pour saisir une description et une position (chiffres uniquement).
- Tous les joueurs reçoivent ensuite une alerte via bulletin, avec la distance et la touche `Y` pour créer un trajet GPS.
